API Documentation

List countries

GET /countries - list all countries with data

GET /countries/country_name - list country with the name specified

POST /countries - list all countries with data

PUT /countries - list all countries with data


Optional Parameter

country_name (string) the Country name used to specify a single country

Example Usage

GET /countries

GET /countries/Australia

POST /countries

PUT /countries


Errors
